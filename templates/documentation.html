API Reference

API provides information about 3 most importanct economical sectors for countries around the world:

  • Industry
  • Agriculture
  • Service

API acepts GET, POST and DELETE types of requests and returns JSON-encoded responses. It doesn't support authentification.

Base URL
http://10.25.100.1:8080

Get countries list

Request

/countries

Request type:
GET
Description
Returns list of countries (name and id)
Response 
{
  "Afghanistan": "5e8ac0e9cbbeb8e072d61e9a",
  ...
}

Get information about specific country

Request

/countries/country_id

Request type:
GET
Description
Returns info about the country with provided id
country_id format
A 24 letters string of digits and letters
Response 
{
  "_id": {
    "$oid": "5e755a5d70c52e15b23f44bf"
  },
  "name": "Albania",
  "data": {
    "service": {
      "1970": "21.8",
      ...
    }
    "industry": {
      "1970": "11",
      ...
    }
    "agriculture": {
      "1970": "67.2",
      ...
    }
  }
}

Update information about specific year for a given country

Request

/countries/country_id

Request type:
PUT
Description
Updates info about country specified by country_id
country_id format
A 24 letters string of digits and letters
Request data format
{
  gdp: '34.5',
  industry: '10',
  agriculture: '24.5',
  service: '54',
  year: '1987'
}
Response

HTTP Status Code

Delete information about specific year for a given country

Request

/countries/country_id

Request type:
DELETE
Description
Deletes info about given year for country specified by country_id
country_id format
A 24 letters string of digits and letters
Request data format
{
  year: '1987'
}
Response

HTTP Status Code

Errors

Possible errors
200 - OK Everything worked as expected.
400 - Bad Request The request was unacceptable, often due to missing a required parameter or mailformed parameter.
404 - Not Found The requested resource doesn't exist.
5xx - Server Errors Something went wrong on the server's end. This is not your fault :)
